Student internships

If you are interested in doing an internship on the topic of quantum algorithms and computing, feel free to contact me at simon.apers@irif.fr. Here are some suggested topics:

  • Quantum algorithms for graph spanners and sparsifiers.

    A graph spanner is a sparse subgraph that approximately preserves shortest-path distances. In a recent work, a quantum algorithm for constructing graph spanners was described. This was used to obtain faster quantum algorithms for a wide range of graph problems such as graph sparsification, computing the minimum cut, and solving Laplacian systems. The aim of this project is to find a faster quantum algorithm for constructing a spanner of an *unweighted* graph. This would lead to faster algorithms for all of the aforementioned problems as well.

    Reference: arXiv:1911.07306.

  • Quantum inspired algorithms for topological data analysis.

    The use of quantum computers for machine learning (ML) is an exciting perspective, and a number of quantum algorithms for ML have been claimed to provide exponential speedups over the best classical algorithms. Somewhat surprisingly, many of these claims were recently disproved by a new flavor of *classical* algorithms that provide similar speedups. One quantum algorithm whose claimed exponential speedup has not been disproven is that for topological data analysis by Lloyd, Garnerone and Zanardi (link). The quantum algorithm uses Hamiltonian simulation and phase estimation to efficiently approximate the Betti number of a simplicial complex. The goal of this project is to take inspiration from the quantum algorithm to investigate classical algorithms for the approximating Betti numbers.

    References: arXiv:2005.02607, Quanta magazine article.

  • Optimal quantum walk search. (co-supervised with F. Magniez)

    Quantum walks on graphs are the quantum analogue of random walks on graphs. They can be used to find marked vertices in a graph quadratically faster than the random walk hitting time. This speedup is similar to the quadratic speedup that Grover search gives for unstructured search problems. Currently, however, there is a tradeoff in the complexity of quantum walk search: either it makes an optimal number of quantum walk steps on the graph, or it makes an optimal number of "checks", where a check tests whether a vertex is marked or not. The goal of this project is to investigate this tradeoff, and ideally show that there is a quantum walk algorithm that is optimal on both accounts.

    References: arXiv:1912.04233, arXiv:quant-ph/0608026.

  • Finding many elements with quantum walk search. (co-supervised with A. Chailloux)

    Quantum walk search was most famously used by Ambainis in his quantum algorithm for element distinctness (link), which amounts to finding a collision of a given input function (i.e., x and y such that f(x)=f(y)). This algorithm is the key building block of a broad range of other quantum algorithms in cryptography, property testing, etc. At its core is a quantum walk algorithm for finding a marked element on the Johnson graph quadratically faster. Such a marked element corresponds to a collision in the input function. A recent work by Chailloux and Loyer (arXiv:2105.05608) raised the interesting question of whether *multiple* collisions can be found more efficiently then just repeatedly running Ambainis' algorithm. The goal of this project is to investigate this question, which more abstractly amounts to investigating the use of quantum walk search for finding many marked elements.

    Reference: arXiv:2105.05608.
